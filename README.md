# angularapp-gulp-boilerplate

A boilerplate to start an Angular webapp with Gulp automated tasks.

## Dependencies
- Angular.js
- Angular-Route
- Angular-Resource
- Bootstrap
- Fontawesome

## Usage
* Install global
```
$ npm install -g stylus
$ npm install -g gulp
$ npm install -g bower
```

* Install local
```
$ npm install
$ bower install
```
